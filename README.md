# PopChat

PopChat is a lightweight random video chat web app inspired by Omegle.

## Features
- Camera & microphone support
- Random peer matching
- Pop Now / Next flow
- Clean UI without chat history pollution

## Tech Stack
- Node.js
- Express
- WebRTC
- WebSockets
- HTML / JavaScript

## Run locally
npm install
node server.js

Open http://localhost:8081
